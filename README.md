# hexo-generator-news

News generator with I18n for [Hexo].

## Installation

``` bash
$ npm install hexo-generator-news --save
```

## Options

``` yaml
index_generator:
  path: ''
  per_page: 10
```

- **path**: Root path for your news index page. (default = '/news')
- **per_page**: Posts displayed per page. (0 = disable pagination)
